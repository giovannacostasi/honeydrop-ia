# HoneyDrop - Inteligência Artificial

Projeto desenvolvido para a disciplina de Inteligência Artificial.

## Descrição
Este projeto propõe o uso de técnicas de Machine Learning para analisar e prever o consumo alimentar de animais em situação de rua, utilizando dados simulados provenientes de um comedouro inteligente.

## Conteúdo
- Dataset (`honeydrop_dataset.csv`)
- Notebook com análise exploratória

## Tecnologias
- Python
- Pandas
- Matplotlib
